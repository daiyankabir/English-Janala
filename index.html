Frequently Asked Questions

1. The difference between var, let, and const? +

var: Old, function-scoped, can be re-declared (avoid using it).
let: Block-scoped, can be updated but not re-declared.
const: Block-scoped, cannot be re-assigned.

2. The difference between map(), forEach(), and filter()? +

forEach() → Just loops through an array, doesn't return anything.
map() → Transforms each item and returns a new array.
filter() → Returns a new array with items that match a condition.

3. Explain arrow functions and how they are different from regular functions? +

Arrow Functions (=>) are shorter and don’t have their own this.
Regular Functions work best when using this inside objects.

4. How JavaScript Promises work? +

A Promise is like a guarantee that you'll get a value later.
It can be Pending → Resolved (then) → Rejected (catch).

5. How closures work in JavaScript? +

A closure is when a function remembers variables from its parent, even after the parent is gone.
